# linkbag

* A bag of links


### Install
```sh
  npm install
```

### Usage
###### Production
```sh
  npm start
```

###### Development
```sh
  npm run watch
```

###### Build
```sh
  npm run build
```


### Notes
* For production, go to http://localhost:3000/
* For development, go to http://localhost:8080/
