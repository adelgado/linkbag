# react-boilerplate

* An Isomorphic React starter project, configured for ES6 React, LESS, Node, Express


### Install
```sh
  npm install
```

### Usage
###### Production
```sh
  npm start
```

###### Development
```sh
  npm run watch
```

###### Build
```sh
  npm run build
```


### Notes
* For producation, go to http://localhost:3000/
* For development, go to http://localhost:8080/
